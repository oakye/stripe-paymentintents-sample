# Name of sample
An [Express server](http://expressjs.com) implementation

## Requirements
* Node v10+
* A configured .env file as discussed in the ../README.txt

## How to run

From the same directory as the server.js file:

1. Install dependencies

```
npm install
```

2. Run the application

```
npm start
```

3. Go to `localhost:4242` to see the demo
